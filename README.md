# fullstack-backend-aiShop
# AI E-commerce Fullstack Project
本项目和仓库里的 fullstack-frontend-aiShop 进行配套使用
一个基于 **React + Zustand + Node.js + Express** 构建的全栈电商项目，支持商品浏览、购物车管理、订单结算，并预留 AI 能力扩展（推荐系统 / 评论总结）。

-  前后端完整打通（非纯前端 demo）
-  Zustand 精细化状态管理（避免不必要渲染）
-  购物车高性能优化（ids + map 结构）
-  支持两种结算模式（立即购买 / 购物车结算）
-  Mock 后端独立实现（可替换真实服务）
-  预留 AI 扩展能力（推荐 / 评论总结）

---
### 前端
- React + TypeScript
- Zustand（状态管理）
- React Router
- SCSS
- Axios（请求封装）

### 后端
- Node.js
- Express
- Mock 数据（内存存储）
